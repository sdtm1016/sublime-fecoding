最流行的前端开发插件，适用于Sublime Text 2/3

Fecoding是专门为sublime text编辑器开发的前端编码插件，适用于Sublime Text 2和Sublime Text 3，如果你也是用Sublime Text作为前端开发工具，你一定不能错过它。

Fecoding基于phython开发，但是核心功能都是通过调用node.js脚本来执行的。作为一名前端开发者，你一定不会对node感觉陌生，Fecoding是一款Sublime Text插件，同时，它堪称是一个插件平台，你可以通过它，来调用你自己编写的node.js脚本。

Fecoding默认内置的插件有如下几种：

CSS格式化

格式化CSS代码，不管是在css文件内，还是在html内，只要是css代码，它都能完美的帮你格式化。

JavaScript格式化

格式化javascript代码，不管是在css文件内，还是在html内，只要是javascript代码，它都能完美的帮你格式化。

如果你想改变格式化代码的规则，你需要修改code format的默认设置：

编译模板

NodeTpl是一款功能强大的前端模板引擎，你一定很熟悉，该插件可以完美地编译NodeTpl文件。

SVN管理

如果你的团队是在用SVN管理代码的，那么，使用该插件，可以非常方便地对文件进行SVN管理，包括提交、新增、更新操作。

图片上传

编辑器能够自动上传页面中的图片文件吗？

没错，正如大家所期待的，在编辑器中可以如我们所愿，自动上传选中代码片段中的图片到服务器，然后替换成上传后的http图片地址。

安装步骤

如果你没有安装 Node.js，请先安装：Node.js

打开编辑器菜单：Preferences -> Package Control -> Install Package，或用快捷键 Ctrl + Shift + P，键入：Install Package

在 Install Package 界面，键入：Fecoding

    

然后回车，安装。

安装完毕后，点击编辑器菜单 Tools -> Fecoding -> Init 进行初始化插件。

然后右键菜单就有所需的 fecoding 插件了
