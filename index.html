Most popular Front-End Developer Plugin for Sublime Text 2/3

Fecoding is a plugin for sublime text 2/3, if you are a front-end developer and coding with sublime text, you may choose it.

Fecoding based on phython, but core lib call node.js. As a front-end developer, you must be familiar to node, Fecoding is a plugin for Sublime Text, also, it's a platform, can call your own node.js scripts. it means, you can write your own plugin for Fecoding.

The default plugin for Fecoding is in the list:

CSS format

Auto format your css code when save. work perfectly in css and html files.

JavaScript format

Auto format your javascript code when save. work perfectly in css and html files.

If you want change any format rules, you can modify the settings file in code format:

Compile Tpl

Compile NodeTpl, a pupular front-end template engine.

SVN

If your team is using SVN managing the code, it'll help you in the editor.

The plugin can help you submit/update/add the file in editor.

Image uploader

Can editor auto upload the images in html?

Yes! We can use this plugin to upload the images to server in your page.

How to install?

If you havn't install Node.js, please install first: Node.js

Open the sublime text menu: Preferences -> Package Control -> Install Package, or use Ctrl + Shift + P, then type: Install Package

In Install Package panel, type: Fecoding

    

It'll install the plugin automatically.

When the install status is ok, click Tools -> Fecoding -> Init to initialize the plugin.

Right click any document in sublime text, you'll see fecoding menu in the contextmenu.
